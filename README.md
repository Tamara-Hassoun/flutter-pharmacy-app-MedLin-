# Flutter Pharmacy App

Mobile application for managing pharmacy tasks. Allows doctor login and medicine selection from the inventory. Built with Flutter using Bloc for state management and backend integration.

# Features

- Doctor login and authentication
- Medicine selection from backend inventory
- Efficient state management using Bloc
- Clean and maintainable code structure

# Technologies

- Flutter (Dart)
- Bloc architecture for state management
- Backend API integration
- UI Design
